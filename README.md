# Streamlit Math

一个基于 Streamlit 构建的数学学习和练习平台。该项目旨在帮助学生更好地理解和掌握数学知识，通过交互式的界面提供数学练习和可视化功能。

## ✨ 功能特点

- 📝 数学公式编辑和显示
- 📊 数学函数可视化
- 🎯 练习题生成和答案检查
- 🎨 清晰直观的用户界面
- 📱 支持移动端访问

## 🚀 快速开始

### 环境要求

- Python 3.7+
- Streamlit
- 其他依赖包

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/streamlit_math.git
cd streamlit_math
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行应用
```bash
streamlit run app.py
```

## 📖 使用说明

1. 打开应用后，选择你想要练习的数学主题
2. 根据界面提示进行相应的操作
3. 查看结果和反馈
